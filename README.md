# UB EDU

Plataforma educacional focada em aprendizado progressivo, acompanhamento de desempenho e prática contínua através de conteúdos multimídia e exercícios interativos.

---

## 🚀 Visão Geral

O UB EDU é uma plataforma que combina:

- 📚 Conteúdo teórico (textos)
- 🎥 Vídeos explicativos
- 🎧 Áudios e podcasts
- 🧠 Exercícios e simulados
- 📊 Acompanhamento de desempenho

Diferente de sistemas tradicionais, o desempenho do usuário não é baseado apenas em acertos, mas também em:

- Conteúdo consumido
- Engajamento com os materiais
- Participação em exercícios

---

## 🧩 Funcionalidades

- Sistema de login e cadastro
- Dashboard de desempenho
- Desafios (exercícios e simulados reutilizados)
- Livro digital com conteúdo estruturado por:
  - Matéria
  - Tópico
  - Subtópico
- Certificados simbólicos por conclusão de matéria
- Sistema de notificações
- Perfil do usuário com histórico
